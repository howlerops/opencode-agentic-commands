Agentic command loops for OpenCode and Pi.

A compact slash-command suite that plans, executes, researches, crafts agents, persists through implementation, reviews diffs, opens secure remote portals, and orchestrates the whole delivery path.

Eight Commands

Short, spellable names with distinct jobs. Use one command directly, or let Polaris coordinate the delivery suite.

/hugin
Plan

Deep context research, story DAGs, risk register, verification matrix, and review gates.

/tyr
Execute

One goal implemented end to end with architecture, verification, critic repair, and summary.

/munin
Research

Baseline, hypothesis, experiment, metric comparison, keep/discard decision, and ledger.

/eitri
Create

Craft opencode-native agents, workflows, commands, skills, or scoped tools from natural language.

/vidar
Persist

Repeat implementation and review repair loops until the task is complete or truly blocked.

/skuld
Review

PR-style findings, targeted fixes, re-verification, and repeated review until clean.

/polaris
Orchestrate

The north-star command: plan, create, research, implement, review, and repair.

/bifrost
Remote

Expose the active OpenCode server for no-gap sync through a secure Cloudflare or ngrok tunnel; separate Web mode is explicit.

Polaris Flow

The full route keeps discovery, implementation, and review connected instead of treating them as separate one-shot prompts.

1. IntakeScope, success criteria, constraints, and optional memory.
2. HuginAnchor plan with evidence and dependency order.
3. EitriCreate agents or workflows only when needed.
4. MuninDefine outcome tests and compare improvements.
5. VidarLoop implementation with repeated Tyr passes.
6. SkuldReview, repair, verify, and review again.

Install

Use the agent-assisted path for the fewest missed steps, or install directly from GitHub, npm, or a local checkout. OpenCode requires a restart after plugin config changes.

Agent Install

Install and configure opencode-agentic-commands by following the instructions here:
https://raw.githubusercontent.com/howlerops/opencode-agentic-commands/refs/heads/main/docs/guide/installation.md

OpenCode

npm install --prefix "$HOME/.config/opencode" github:howlerops/opencode-agentic-commands
node "$HOME/.config/opencode/node_modules/opencode-agentic-commands/scripts/install-opencode.mjs"

# local checkout
node scripts/install-opencode.mjs

Pi

pi install git:github.com/howlerops/opencode-agentic-commands

# then try
/polaris Ship the feature end to end
One package, two runtimes. This page covers both OpenCode and Pi because they install the same command suite. The full raw guide is available at docs/guide/installation.md for LLM agents and manual troubleshooting.
